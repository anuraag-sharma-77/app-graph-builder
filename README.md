# App Graph Builder

A React + TypeScript application for visualizing application architecture using React Flow.

## Features

- Interactive graph visualization
- Node selection
- Inspector panel
- Config and Runtime tabs
- Zustand state management
- TanStack Query data fetching
- Mock API integration
- Responsive layout

## Tech Stack

- React
- TypeScript
- Vite
- React Flow
- Zustand
- TanStack Query

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Architecture

Mock API
↓
TanStack Query
↓
React Components
↓
Zustand UI State
↓
React Flow Canvas
